# Meow Moments

Приложение для ежедневного получения интересных фактов о котах с красивыми изображениями.

## Основные функции

- Ежедневный факт о котах
- Коллекционирование фактов
- Календарь собранных фактов
- Офлайн-режим

## Технологии

- Kotlin
- Jetpack Compose (View System)
- MVVM Architecture
- Hilt (Dependency Injection)
- Room (Local Database)
- Retrofit (Network)
- Coil (Image Loading)

## API

- [Cat Facts API](https://alexwohlbruck.github.io/cat-facts/) - для получения текстовых фактов
- [Cataas](https://cataas.com) - для получения изображений котов

## Установка

1. Клонируйте репозиторий
2. Откройте проект в Android Studio
3. Синхронизируйте Gradle
4. Запустите приложение на устройстве или эмуляторе
